# Código de Conduta

## Nosso Compromisso

Nos comprometemos a tornar a participação em nosso projeto uma experiência livre de assédio para todas as pessoas, independentemente de idade, aparência física, deficiência, etnia, identidade ou expressão de gênero, nível de experiência, nacionalidade, religião ou orientação sexual.

## Nossos Padrões

Exemplos de comportamentos que contribuem para um ambiente positivo incluem:

- Usar linguagem acolhedora e inclusiva;
- Respeitar diferentes pontos de vista e experiências;
- Aceitar críticas construtivas com maturidade;
- Demonstrar empatia com os outros participantes;
- Colaborar de forma ética e profissional.

Exemplos de comportamentos inaceitáveis incluem:

- Uso de linguagem ofensiva ou discriminatória;
- Ataques pessoais ou políticos;
- Assédio público ou privado;
- Publicação de informações privadas sem permissão;
- Qualquer conduta considerada inadequada em um ambiente profissional.

## Responsabilidades de Aplicação

Os responsáveis pelo projeto têm a responsabilidade de esclarecer e aplicar os padrões de comportamento aceitável.

Eles podem remover, editar ou rejeitar comentários, commits, issues, pull requests ou outras contribuições que não estejam alinhadas com este Código de Conduta.

## Escopo

Este Código de Conduta se aplica a todos os espaços do projeto, incluindo repositórios, issues, pull requests, discussões, documentação e demais canais oficiais relacionados ao projeto.

Também se aplica quando uma pessoa representa oficialmente o projeto em eventos, redes sociais ou outros ambientes públicos.

## Aplicação

Casos de comportamento abusivo, ofensivo ou inaceitável podem ser reportados aos responsáveis pelo projeto.

Todas as denúncias serão analisadas de forma justa e respeitosa.

## Diretrizes de Aplicação

As consequências para violações deste Código de Conduta podem incluir:

1. **Correção**: orientação privada sobre o comportamento inadequado.
2. **Advertência**: aviso formal sobre a violação.
3. **Suspensão temporária**: afastamento temporário das interações no projeto.
4. **Banimento permanente**: remoção definitiva da participação no projeto.
